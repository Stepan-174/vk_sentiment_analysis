# vk_sentiment_analysis
Cистема сентимент-анализа информационного поля в социальной сети ВКонтакте, которая проводит сбор публикаций и комментариев, а затем комплексный анализ тональности (позитивной, нейтральной и негативной) постов и комментариев сообществ ВКонтакте, выявляет ключевые темы обсуждения, оценивает соответствие эмоциональной окраски комментариев постам, а также формирует аналитический отчет о настроениях аудитории.

pipeline: VK → сбор постов → обработка текста → анализ тональности → аналитический отчет

Проект выполнен на Python.

Требуются библиотеки: vk_api nltk, transformers, torch, numpy, pandas, matplotlib, seaborn scikit-learn, keybert.

Загрузить через командную строку командой: pip install vk_api nltk transformers torch numpy pandas matplotlib seaborn scikit-learn keybert

Так же необходимо получить токен VK API

Файлы в директории:

Запуск в Windows в командной строке: python test3.1.py & python test7.1.py

test3.1.py – содержит скрипт, который запускает сбор, очистку и предварительный сентимент-анализ текста и записывает данные в файлы discourse.txt и sentiment_analysis.txt.

ВНИМАНИЕ: в  файле test3.1.py TOKEN = 'c64'  # нужно заменить на свой токен. Иначе работать не будет.

test7.1.py – содержит скрипт, который запускает создание аналитического отчета и визуализации, а затем сохраняет полученные результаты в файлы resume.txt и visualization.png.

discourse.txt – содержит необработанный текст, собранный из постов и комментариев в формате JSON.

sentiment_analysis.txt - содержит обработанный текст, собранный из постов и комментариев с предварительно обработанным сентимент-анализом в формате JSON.

resume.txt – содержит отчёт по проведённому анализу тональности.

visualization.png – содержит визуализацию по проведённому анализу.

### Исходные данные

Формат: система формирует JSON-массив с записями постов, содержащими текст, идентификаторы группы и поста, даты, метаданные (лайки, репосты, просмотры, количество комментариев) и вложенный список комментариев с их тональностью.

### Основные этапы работы

1. Парсинг и предобработка JSON-файла.

2. Подсчёт общего числа постов и комментариев.

3. Анализ распределения тональностей постов и комментариев.

4. Вычисление процентного соотношения позитивных, нейтральных и негативных сообщений.

5. Сравнительный анализ эмоциональной окраски постов и ответных комментариев.

6. Автоматическое выделение ключевых тем обсуждений с помощью метода KeyBERT.

7. Определение 10 наиболее значимых негативных постов по числу комментариев для приоритетного рассмотрения.

8. Визуализация результатов распределения тональностей и сравнительного анализа с помощью графиков.

9. Формирование итогового отчёта в текстовом файле resume.txt и графического файла visualization.png.

Система представляет результаты в следующем виде: 

Общее количество проанализированных постов: X

Общее количество комментариев: Y

Распределение тональностей постов:
 
 - Positive: количество (%)
 
 - Neutral: количество (%)
 
 - Negative: количество (%)

Распределение тональностей комментариев:
 
 - Positive: количество (%)
 
 - Neutral: количество (%)
 
 - Negative: количество (%)

Процент совпадения тональностей постов и комментариев составляет ZZ.ZZ%.

Интерпретация: Тональности комментариев в значительной степени соответствуют эмоциональной окраске постов (или другое — в зависимости от данных).

### Ключевые темы обсуждений

Среди основных тем, выделенных методом KeyBERT, обнаружены следующие ключевые слова и фразы: (перечисляет ключевые темы через запятую)

10 самых негативных постов

Выявляет 10 постов с негативной тональностью и самым большим количеством комментариев, которые, вероятно, вызывают наибольший интерес сообщества или спорные ситуации. Для каждого из них представлены: текст, ID сообщества, ID поста, URL-ссылка на пост и количество комментариев. Эти материалы полезны для оперативного мониторинга и принятия мер по модерации или ответным действиям.

### Визуализация

Графики распределения тональностей постов и комментариев, а также сравнительный анализ эмоциональной окраски визуализированы и сохранены в файле visualization.png.

Первый график демонстрирует численное распределение постов по трем категориям тональности.

Второй — аналогичное распределение для комментариев.

Визуальный анализ подтверждает данные таблицы и помогает более эффективно воспринимать результаты.

Выводы и рекомендации зависят от результатов анализа.
